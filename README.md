# 📊 E-Commerce Data Analysis

## 📌 Overview
This project analyzes e-commerce sales data to extract meaningful insights and support business decision-making.

---

## 🛠️ Tools & Technologies
- Python (Pandas, NumPy)
- Matplotlib, Seaborn
- SQL (MySQL)
- Jupyter Notebook

---

## 🧹 Data Cleaning
- Handled missing values (mean/median/mode)
- Removed duplicates
- Cleaned invalid values (e.g., 'abd', '4a')
- Converted data types
- Fixed date format
- Created new column: **Total = Price × Quantity**

---

## 📊 Key Insights
- Revenue is mainly driven by product price (high correlation)
- Books category generates highest revenue
- Cash on Delivery is most used payment method (~40%)
- High return rate indicates operational issues
- Revenue shows strong growth over time (2025 peak)
- Sales show seasonal patterns (Feb & Oct peaks)

---

## 📈 Analysis Performed
- Product-wise revenue
- Category-wise analysis
- Payment method distribution
- Order status analysis
- Monthly & yearly trends
- Correlation analysis

---

This project highlights how data cleaning and analysis can uncover valuable business insights and help improve performance.
